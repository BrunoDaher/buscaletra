Formulários

Formulários são utilizados pra ler dados da página e enviar, ou
mesmo executar cálculos.

Elementos do Form Form - Formulário para entrada de dados
TextField - Campo para texto
TextArea - Campo para texto grande
Button - Botão de ação
Label - Rótulo para inputs
Input - Campo para entrada de dados
Inputs
  • type="number" - formato adequado para números
    Idade
  • type="search" - ideal para pesquisas - pode conter dados de uma lista
    Pesquisa
  • type="password" - ideal para pesquisas - pode conter dados de uma lista
    Idade
  • type="url" - aceita somente caminhos url
    Url
  • type="range" - uma barra deslizante
    Volume
Sob ação do mouse o Label direciona o cursor do mouse para o input / textField/ textArea à qual faz referência.
Uma lista completa de ações pode ser encontrada em https://developer.mozilla.org/.
Seletores São propriedades de um elemento que o identificam para o navegado. "id" , "class" , são seletores.
Com JavaScript é possível atribuir funções aos elementos e manipulá-los.
  1. Id - representação "#"
  2. class - representação - "."
Exemplo
<input class='customInput' id='nomeA'>

No código o seletor id recebe o valor "nomeA", e o seletor class recebe o valor "customInput". o JavaScript pode atribuir uma função específica a este input, ou mesmo ler os valores nele contido através do seletor id. O navegador lê o código HTML e o CSS, e se utiliza dos seletores class ou mesmo do id no momento de renderizar a página. Caso haja alguma referência ao elemento HTML nó código CSS, desde que corretamente escrito, o navegador processa e escreve respeitando as regras definidas, como regras de cor, fonte e outras mais.

Em código JavaScript var x = document.getElementById('nome');

O código acima atribui à variavel X o elemento input , cujo id é "nome". A partir disso é possível realizar operações diversas.

Em código Css .customInput {color:blue;font-size:10px;}

O código acima atribui à todo elemento que pertencer à "customInput" as características escritas dentro das chaves, como color e font-size, que são atributos de css.

Os atributos Method e Action são exclusivos do objeto form, cujas funções são enviar dados do formulário. O método deste envio pode ser POST ou GET, ou demais que serão descritos no tópico REST.
